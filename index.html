The Slider

What is this?

Adapt.js is a lightweight (848 bytes minified) JavaScript file that determines which CSS file to load before the browser renders a page. If the browser tilts or resizes, Adapt.js simply checks its width, and serves only the CSS that is needed, when it is needed. 960gs is released under GPL and MIT license.

Flexslider is probably one of the best responsive slider out there, try it, it will blow your mind and guess what, it's under the MIT license!.

rLightbox development is discontinued at the moment but it's a damn good responsive lightbox and it's also under the MIT license :).

TinyNav is a cool responsive menu that will transform a list of link (navigation) into a select box for mobile device at low size. It rocks and it's also released under the MIT license!
